Github token generálás

  1. Profil → Settings
  2. Bal alsó: Developer Settings → Personal access tokens
  3. Tokens (classic) → A lenyílóban újra rákattintunk: Generate token classic
  4. Megjegyzés, hogy mi ez a token és lejárati idő beállítása
  5. Mindent kipipálunk (biztos ami biztos) → Generate token
